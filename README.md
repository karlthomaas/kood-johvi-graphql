## Run Locally

Clone the project

```bash
  git clone https://github.com/karlthomaas/kood-johvi-graphql.git
```

Go to the project directory

```bash
  cd kood-johvi-graphql
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
